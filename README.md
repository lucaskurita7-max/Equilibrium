# Equilibrium — Pro Start (Demo)

Este pacote é uma **demo profissional** para subir rápido no GitHub/Vercel e testar o fluxo:
- Login (sem senha, email demo)
- Listagem de pacientes (multi-tenant)
- Criar paciente
- Ficha do paciente (metas + plano alimentar)
- Exportação por impressão (gera PDF no celular/PC)

## Rodar local

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000

## Deploy Vercel (sem técnico)

1. Suba este projeto no GitHub
2. Na Vercel: New Project → Import → Deploy
